# Cheias no Rio Grande do Sul

Base de dados e informações geográficas na Região Hidrográfica do Lago Guaíba de 2024

## Introdução

Este projeto tem a finalidade de analisar o trabalho feito pela Universidade Federal do Rio Grande do Sul (UFRGS) em conjunto com pesquisadores
voluntários para entender como as cheias do lago Guaíba afetaram as famílias gaúchas.
<br>
Para saber mais sobre a pesquisa feita pela UFRGS, acesse esse [link](https://storymaps.arcgis.com/stories/a81d69f4bccf42989609e3fe64d8ef48)

## Disclaimer

A plataforma criada no projeto é sem fins lucrativos, somente utilizado para análise.
